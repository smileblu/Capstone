# Documentation

이 폴더는 본 프로젝트의 **부가 설명 문서(Documentation)** 를 정리하기 위해 구성되었습니다.  
코드만으로는 이해하기 어려운 **설계 의도, 데이터 흐름, 알고리즘 개념, 시나리오 정의** 등을 문서화하는 것을 목표로 합니다.

## Purpose
- 프로젝트 전반의 구조와 설계 배경 설명
- 탄소 배출 산정 및 절감 시나리오 로직에 대한 보조 설명
- 팀원 간 이해 공유 및 향후 유지보수를 위한 기록

## Contents
이 폴더에는 다음과 같은 문서들이 포함될 수 있습니다.

- **Carbon Calculation Logic**  
  개인·학교·중소기업 활동 데이터를 기반으로 한 탄소 배출량 산정 방식 설명

- **Baseline & Forecast Definition**  
  기준선(baseline) 설정 방법과 시계열 예측 개념 정리

- **Emission Reduction Scenario**  
  AI 기반 절감 시나리오 추천 로직 및 시나리오 유형 설명

- **Data Flow & Processing**  
  사용자 입력 → 데이터 검증 → 보정 → 공공 데이터 매핑 → 결과 산출까지의 흐름

- **System / AI Overview**  
  모델 구조, 사용한 알고리즘, 설계 선택 이유 등 개념 중심 설명

## Notes
- 본 폴더의 문서는 **코드 구현이 아닌 개념과 설계 설명**에 초점을 둡니다.
- 세부 구현 내용은 각 기능별 소스 코드와 함께 참고하는 것을 권장합니다.
